About

Aim of bootstrap Essentials extension is to overcome technical difficulty in responsiveness by using classes designed for specific breakpoints. Not only this project standardize spacing/avatar/container classes, it also drastically reduces the amount of custom css overwrittern above vanilla bootstrap.

This extension has been developed by understanding various templates and layouts used to develop site. Classes have been developed such that your bootstrap theme can evolve with little or no css modification.
